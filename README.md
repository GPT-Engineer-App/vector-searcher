# vector-searcher

A vector search engine for efficiently searching and retrieving items based on vector similarity. Ideal for applications in recommendation systems, image search, and natural language processing tasks. The interface will be clean and minimalist, focusing on user-friendly search functionalities.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/vector-searcher.git
cd vector-searcher
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
